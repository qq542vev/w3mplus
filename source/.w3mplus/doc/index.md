# w3mplus モード {#w3mplus-mode}

w3mplus モードは Pentadactyl + Pale Moon 風のキーマップです。

w3mplus モードは [Pale Moon モード](#palemoon-mode)の上に構築されています。上書きされていない Pale Moon モードのキーも利用できます。
